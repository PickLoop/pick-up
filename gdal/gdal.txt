Building on Windows instal step
教程：http://blog.csdn.net/q_l_s/article/details/52468309 
1. 下载源码：http://download.osgeo.org/gdal/
2. 编译工具为Visual Studio自带工具: VS2012 x86 Native Tools Command Prompt
3. 编译命令
	4.1 nmake /f makefile.vc  //配置文件，需要点时间，注意是否提示有问题，如果有问题，重新配置makefile.opt
	4.2 nmake /f makefile.vc install //生成bin/html/data文件夹，也可能出现错误，同样重新配makefile.opt
	4.3 nmake /f makefile.vc devinstall //生成lib/include文件夹
4. nmake.opt配置
	2.1 GDAL Home = "D:\gddal"  //编译输出结果
	2.2 MSVA_VER=1800  //Visual Studio版本，1800对应VC11
	2.3 支持GEOS
		2.3.1 # Uncomment for GEOS support 后三行注释打开 GEOS_DIR、GEOS_CFLAGS、GEOS_LIB
		2.3.2 GEOS_DIR后的内容为geos的本地路径
		2.3.3 ”GEOS_CFLAGS=“ 和 “GEOS_LIB” = 后的‘/’ 改为 '\'
